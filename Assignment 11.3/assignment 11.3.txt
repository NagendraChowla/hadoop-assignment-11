Step 1: Login to twitter.

Step 2: From twitter app get all the four required keys.

Step 3: Create a flume.conf file under /usr/local/flume/conf directory.

Step 4: Load source, sink, memory channels, four keys from twitter app, keywords to fetch tweets and other details in flume.conf. 

Step 5: execute the command - flume-ng agent -n TwitterAgent -f '/usr/local/flume/conf/flume.conf'

A file will be created in  /user/flume/tweets/ which contains tweets generated.
